# 🌍 Travelgram – Página de Perfil de Viagens

Este projeto foi desenvolvido como parte do curso **Full Stack** da **Rocketseat**.

O **Travelgram** é uma página de perfil de viagens com foco em praticar conceitos básicos de **HTML e CSS**, simulando uma rede social onde é possível exibir destinos visitados imagens das viagens.

## 🧪 Tecnologias

- HTML5
- CSS3

## 💡 Funcionalidades

- Exibição de perfil com nome e foto do usuário
- Lista de destinos com imagens
- Layout responsivo básico
